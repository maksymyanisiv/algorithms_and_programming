Варіант 31 — Янісів Максим

Умова задачі:

Обчислити і вивести таблицю значень F(x) на інтервалі [Xпоч, Xкін] з кроком dX

F = a·x² + b/c, якщо x < 1 і c ≠ 0
F = (x-a)/(x-c)², якщо x > 1.5 і c = 0
F = x²/c², в інших випадках

де a, b, c – дійсні числа

4 способи (цикли):
1. while (…) { … }
2. do { … } while (…);
3. for (…; …; x+=dx) { … }
4. for (…; …; x-=dx) { … }

Блок-схеми алгоритмів

F(x, a, b, c) - умовна функція

1. while - табулювання

2. do-while - табулювання

3. for (x+=dx) - табулювання

4. for (x-=dx) - табулювання

UML-діаграми дій

F(x) - UML Activity

while - UML Activity

do-while - UML Activity

for x+=dx - UML Activity (loop)

for x-=dx - UML Activity (loop)
