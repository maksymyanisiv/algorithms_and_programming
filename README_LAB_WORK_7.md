# Лабораторна робота №7
## Робота з базами даних в табличному процесорі

### Створення файлу

Запустіть скрипт для створення Excel файлу:
```bash
python3 solve_lab_work_7.py
```

Це створить файл `Yanisiv.xlsx` з усіма необхідними листами та таблицями.

### Структура файлу

1. **Лист "Зарплата"** - таблиця зарплат працівників ІТ з автоматичним розрахунком податку та суми до видачі
2. **Лист "Зарплата (формули)"** - копія таблиці з відображенням формул
3. **Лист "Статистика ІТ"** - таблиця статистики ІТ-галузі в Україні
4. **Лист "Автофільтр"** - копія таблиці статистики для роботи з автофільтром
5. **Лист "Розширений фільтр"** - таблиця діяльності працівників фірми ІТ-Market@

### Виконання завдань з сортування та фільтрації

#### Завдання 8-9: Сортування на листі "Статистика ІТ"

1. Відкрийте лист "Статистика ІТ"
2. Виділіть діапазон даних (A3:E18)
3. Перейдіть до **Дані → Сортування**
4. Для завдання 8: Сортувати за стовпцем "Кількість ФОП-чоловіки" (спадання)
5. Для завдання 9: Сортувати за стовпцем "Область" (зростання)

#### Завдання 11-13: Автофільтр на листі "Автофільтр"

1. Відкрийте лист "Автофільтр"
2. Автофільтр вже встановлено (стрілочки біля заголовків)
3. **Завдання 11**: Натисніть на стрілку біля "Область" → Текстові фільтри → Починається з → Введіть "Ч"
4. **Завдання 12**: Натисніть на стрілку біля "Сума сплачених податків ФОП-ІТ" → Числові фільтри → Більше → Введіть 500000
5. **Завдання 13**: Натисніть на стрілку біля "Відсоток ФОП-чоловіки (%)" → Числові фільтри → Між → Введіть 20 та 40
6. **Завдання 14**: Натисніть на стрілку біля будь-якого стовпця → Очистити фільтр

#### Завдання 16-19: Розширений фільтр на листі "Розширений фільтр"

1. Відкрийте лист "Розширений фільтр"
2. Створіть область критеріїв (наприклад, у стовпцях H-J)
3. **Завдання 16**: Знайдіть найбільш поширене ім'я (Іван - 4 рази), створіть критерій для імені "Іван"
4. **Завдання 17**: Критерій: Прізвище = "Василенко"
5. **Завдання 18**: Критерій: Прізвище починається з "Б" або "П" (використайте умови "Б*" та "П*" в окремих рядках)
6. **Завдання 19**: Критерій: Внесок у розвиток фірми між 500 та 900

### Примітки

- Всі формули вже налаштовані та працюють автоматично
- Податок розраховується за правилами: 10% до 8000 грн, 20% більше 20000 грн, 15% в інших випадках
- Відсоток ФОП-чоловіків розраховується автоматично
- Для перегляду формул на листі "Зарплата (формули)" натисніть Ctrl+` або перейдіть до Формули → Показати формули
