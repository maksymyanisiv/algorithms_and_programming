# Розв'язання контрольних робіт з лінійної алгебри та аналітичної геометрії

Скрипт `solve_control_work.py` призначений для автоматичного розв'язання задач контрольних робіт з лінійної алгебри та аналітичної геометрії.

## Встановлення залежностей

```bash
pip3 install numpy
```

## Використання

### Запуск для конкретного варіанту

```bash
python3 solve_control_work.py <номер_варіанту>
```

Приклад:
```bash
python3 solve_control_work.py 1
```

### Доступні варіанти

Скрипт підтримує розв'язання всіх 20 варіантів контрольних робіт (варіанти 1-20).

## Типи задач, які розв'язує скрипт

### 1. Системи лінійних рівнянь
- Розв'язання систем з 3 невідомими
- Використовує метод Гаусса через `numpy.linalg.solve`

### 2. Обернені матриці
- Обчислення оберненої матриці
- Підтримка матриць 3x3
- Перевірка на виродженість

### 3. Векторні операції
- Скалярний добуток векторів
- Векторний добуток
- Обчислення кута між векторами
- Лінійні комбінації векторів

### 4. Аналітична геометрія
- Відстань від точки до прямої (2D)
- Рівняння прямої через точку паралельно/перпендикулярно вектору
- Рівняння прямої через дві точки
- Нормальний вигляд рівняння прямої
- Рівняння площини через три точки
- Рівняння площини через точку перпендикулярно вектору
- Відстань між паралельними площинами
- Перевірка паралельності/перпендикулярності площин

### 5. Криві другого порядку
- **Еліпс**: канонічне рівняння, властивості (фокуси, вершини, ексцентриситет, директриси)
- **Гіпербола**: канонічне рівняння, властивості (фокуси, вершини, ексцентриситет, асимптоти)
- **Парабола**: канонічне рівняння, властивості (фокус, директриса)
- **Коло**: центр, радіус, канонічне рівняння

## Структура коду

Скрипт містить:

1. **Базові функції** для математичних операцій:
   - `solve_linear_system()` - розв'язання систем рівнянь
   - `inverse_matrix()` - обернена матриця
   - `matrix_power()` - піднесення матриці до степеня
   - `dot_product()`, `cross_product()` - векторні операції
   - `vector_angle()` - кут між векторами
   - Функції для роботи з прямими та площинами
   - Функції для кривих другого порядку

2. **Функції для кожного варіанту**:
   - `solve_variant_1()` до `solve_variant_20()`
   - Кожна функція розв'язує всі 5 задач свого варіанту

```

Варіант 1

1) Система рівнянь:
Розв'язок: x = 0.1667, y = 1.1333, z = -3.9333

A^(-1) =
[[-3. -1.  2.]
 [ 2. -1.  0.]

3) Векторні операції:
5a - 2b = [-4  2 32]
a·(3b) = 6.0000
(2a) × b = [-52  24  -8]
Кут між a і 2b: 86.0431°

4) Відстань та площина:
Відстань від A(-2, 3) до прямої 3x - 4y - 2 = 0: 4.0000
Рівняння площини: -5.0000x + 7.0000y + 3.0000z + 16.0000 = 0

5) Канонічне рівняння еліпса:

```

## Примітки

- Деякі задачі потребують додаткових даних, які не були чітко вказані в умові. У таких випадках виводиться повідомлення "потрібні додаткові дані".
- Всі результати виводяться з точністю до 4 знаків після коми.
- Скрипт використовує бібліотеку NumPy для точних математичних обчислень.
